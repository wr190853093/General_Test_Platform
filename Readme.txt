通用测试平台功能目标：
1、实现接口自动化测试
2、实现UI自动化测试

主要功能模块包括：
组织管理：
        管理公司组织机构：新增、编辑、删除
        短期目标：录入机构，为人员提供节点
用户管理：
        管理用户：新增、编辑、启用、禁用
        短期目标：人员登录，预留必要字段。
        长期目标：使用预留字段实现权限管理，邮件等功能。
项目管理：
    管理项目：新增，编辑，团队管理
模块管理
    管理项目内功能模块：新增，编辑，
环境管理
    配置项目环境：实现开发环境，测试环境配置
接口测试
    接口管理
        管理各模块接口：新增，编辑，参数配置，启用禁用
    用例管理
        组织接口测试用例：数据配置、预期结果（断言），前置处理，后置处理，执行
    执行管理
        执行用例集，生成报告
    测试报告
        查看执行结果报告
mock管理
    mock数据管理
        使用接口测试编写的数据，进行mock数据编辑，提供mock服务
